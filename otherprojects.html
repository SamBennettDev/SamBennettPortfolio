Web Server

This project revolved around a building a Web Server that is able to store and retrieve information from a database, and produce simple web pages to show this information via a web browser. I cannot post the code as it is an assignment that will be used for future students.

Learning Outcomes

  • How to use SSMS or Visual Studio to directly interact with a Database Server and databases thereon.
  • The basic syntax of information insertion and retrieval from a an SQL database.
  • How to connect to an SQL Server and send SQL "commands" to it, all via the C# language.
  • How to define and create simple relationships/tables using an SQL management program.
  • The basics of the HTTP protocol.
  • How to construct basic web pages to display tabular (or figure) data.
  • Set up and use a secret config file.
