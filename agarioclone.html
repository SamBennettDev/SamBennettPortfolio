Agar.io
Clone

About

This project contains the client code for the agar.io clone under a time restraint of one week. This project was written in C# using windows forms and visual studio IDE. I unfortunately cannot include my source code for the assignment as this assignment will be given to future students. I have included a video showcasing the functionality of the program below.

Specifications

  • Structured following a MVC architecture.
  • Connect to given game server.
  • Create a Core Library Project containing World class, GameObject class, Player class, and Food class.
  • Create the GUI which includes a welcome screen and the game screen.
  • Render the world including all the food and players from the data sent by a server.
  • Send move and split requests to the server.
  • Use JSON to serialize and deserialize objects.

Learning Outcomes

  • How to create JSON from objects (serialize) and how to create objects from JSON (deserialization).
  • How to employ networking code and a networking library to build a client/server project.
  • How to deal with multiple threads (GUI + (one or more) Networking) including data protection
  • How to build a reasonably complicated GUI including understanding applying "Math" to "Coding".
  • How to communicate with stake holders and ask appropriate questions about functionality.
